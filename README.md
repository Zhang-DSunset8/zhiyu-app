# 情绪果园

一款以「接纳而非任务」为核心理念的情绪疗愈前端应用。

## 功能概览

- **果园** — 情感浇灌机制，果树成长与收获，种子选择器，洞悉日历
- **画室** — Canvas 绘画工具 + 画廊管理
- **冥想** — 分类课程、系列冥想、底部播放器（接纳式退出）
- **发现** — 心情日记、自我关怀、官方专题
- **我的** — 个人资料、成就系统、偏好设置、帮助中心、数据备份/恢复

## 快速开始

```bash
npm install
npm run dev
```

浏览器访问 `http://localhost:5173`

## 技术栈

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Zustand（本地持久化）

## 数据存储

所有用户数据保存在浏览器 `localStorage` 中，支持 JSON 导出/导入备份。
